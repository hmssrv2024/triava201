Instalar aplicación

No pudimos abrir la descarga automática

Algunos navegadores antiguos no muestran el mensaje de instalación. Puedes añadir la aplicación manualmente siguiendo estas instrucciones:

Android (Chrome, Edge, Brave)

  1. Pulsa el botón de menú ⋮ del navegador.
  2. Elige Añadir a pantalla de inicio o Instalar aplicación.
  3. Confirma la instalación para crear el acceso directo.

iPhone o iPad (Safari)

  1. Toca el icono Compartir (cuadrado con flecha).
  2. Selecciona Añadir a pantalla de inicio.
  3. Confirma con Añadir para finalizar.

Instalación disponible solo en Android

Safari no admite la instalación automática de la aplicación. Usa un dispositivo Android para descargarla y continuar con tu registro.
